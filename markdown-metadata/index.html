Because frontmatter breaks Markdown syntax and puts the title at the same importance as other metadata like the date and thumbnail.

The title is a primary piece of data for the document, and it is not on the same level of importance as the date.

With frontmatter, your Markdown (not metadata) document starts with a level 2 heading and no level 1 heading, or it has multiple level 1 headings (titles).

Instead, put the document heading at the top using Markdown syntax, and add metadata like the date below that.
