Dark theme

I've put in a dark theme that triggers automatically if the browser detects you're using a dark theme, so you might not actually see the black on white.

Font selection

I decided to forgo the helvetica-ish font, since it's not that great for a lot of reading, and I instead chose Atkinson Hyperlegible, which I've paired with DM Serif Display for the headings.

The cool thing about Atkinson Hyperlegible, is that it's designed for low-vision people, in collaboration with the Braille Institute.

DM Serif is just there to look nice.
