If The Web Was Simpler

If the only point of a web page was to serve text, then there might be no harm in not having any default styling, and letting the various web browsers theme the content in the way that they deem most appropriate.

This might be the beginning of a uniform web. You would view the web through the lens of your browser, or your custom stylesheet. Each website you visit would not flip the paradigm on you. That would be interesting, no?
