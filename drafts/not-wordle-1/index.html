Step juan: the page🔗

Start with a display element, which is where all the fun stuff will happen, then add an input below it, and you’re done! Maybe add some basic instructions, if you want.

<div id=display></div>
<input type=text id=guess placeholder="Guess here"></input>
<button>Submit</button>

Oh, and a title.

<h1>Ceci n'est pas Wordle</h1>

Step two: make the cells visible🔗

Add a bit of CSS to make sure all your empty cells don’t just collapse into nothing — set a width and height for them.

Step three: sprinkle javascript🔗

Start by setting your word. We’ll go with ‘hello’ (duh).

Then we have to draw the grid.
Select your display and make 6 rows (each row is a guess), and a cell for each letter of the word.

'use strict';

const WORD = 'hello';

const display = document.getElementById('display');

for ( let i = 0; i < 6; i++ ) {
  const row = document.createElement('div');
  for ( let j = 0; j < WORD.length; j++ ) {
    const cell = document.createElement('div');
  }
}
